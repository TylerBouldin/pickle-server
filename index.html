Welcome to the Pickleball API!

This API provides access to pickleball products, court information, and community groups. All endpoints return JSON data and support CORS for easy integration with your applications.

Base URL: http://localhost:3001 (Local) or https://your-render-url.onrender.com (Production)
üè™ Products
GET /api/products
Retrieve all pickleball products including paddles, balls, shoes, and accessories.
Returns:
Array of 8 products with id, name, image, price, description, skillLevel, and detailedDescription
GET /api/products/:id
Retrieve a specific product by its ID (1-8).
Example:
/api/products/1 returns the Pro Paddle Set
